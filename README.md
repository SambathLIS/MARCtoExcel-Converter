# MARC to Excel Converter (Advanced Preview Tool)

A lightweight **Python + Tkinter desktop application** for librarians and metadata professionals to:

- Upload MARC (.mrc, .xml, MARCXML) files
- Preview records **tag-series wise (0XX–9XX)**
- View **every MARC tag and every subfield**
- Filter tags and subfields
- Export clean, structured data to **Excel, CSV, or TXT**

No PyQt / No DLL issues — works out of the box with Python.

---

## ✨ Features

✔ Supports `.mrc`, `.xml`, `.marcxml`  
✔ Tag series preview: **0XX–9XX**  
✔ Every MARC tag & every subfield exported  
✔ Repeated subfields handled (`245$a_1`, `245$a_2`)  
✔ Preview limit control  
✔ Tag / Subfield filtering  
✔ Export to:
- Excel (`.xlsx`)
- CSV (`.csv`)
- TXT (`|` delimited)

✔ Built using **Tkinter** (no external GUI dependencies)
